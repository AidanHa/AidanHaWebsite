RoastMyResume.ca

Full-Stack Resume-Roasting Website

This project is a full stack website that allows users to register an account, upload their resume, and post comments on other resumes. I handled user data and implemented a REST API with back-end functionalities using Node.js, Express.js and MongoDB. There were also various cloud-based services implemented within the website to reduce stress, including AWS to host the database and Heroku to host the website.

  • Date Created: July 1 2018
  • Date Finished: September 14 2018
  • Category: Full Stack, Web Dev

Fidelity Investing App

Fidelityh Canada Case Competition

For this project, I created a discount brokerage Android app for Fidelity Canada's Case Competition in order to promote investing within millennials. The Botaneer

Automatic Plant Watering System

For this project, our team developed a C++ program that continuously reads a humidity and temperature sensor, then powers a DC pump to water a plant if the humidity is below a certain threshold. We also SSH into theOmega2 CPU to control the system's functionality, used Vim in the Omega's Linux environment to edit code, and implemented Docker to cross compile custom libraries

  • Date Created: November 1 2017
  • Date Finished: December 3 2017
  • Category: Software Dev, Hardware Dev, C++
